# Milestone A — Demo Script

## Prerequisites

```bash
# Terminal 1: Start Supabase
npx supabase start
npx supabase db reset

# Terminal 2: Serve Edge Functions
npx supabase functions serve --env-file .env

# Terminal 3: Start Expo
npx expo start
```

Open the app on a device or simulator.

## Demo Flow (3-5 minutes)

### 1. First Launch (30s)
- App opens to **onboarding intro** screen
- Show: dark theme, LEVEL branding, "Your real-life game begins now"
- Tap **"Get Started"**
- Note: anonymous auth happens in background — no sign-up friction

### 2. Onboarding (60s)
- **Vibe Selection**: Tap 2-3 vibes (e.g., Fitness + Adventure)
  - Show: haptic feedback on tap, cards highlight with brand color
  - Tap **"Next"**
- **Duration Selection**: Tap "Steady (15 min/day)"
  - Tap **"Next"**
- **Starter Pack**: Show the Day 1 Kit summary
  - Tap **"Let's Go!"**
  - Note: loading spinner while bootstrap function runs
  - App navigates to home with real daily quests

### 3. Home Screen (30s)
- Show: **3 daily quest cards** with real titles from the seed data
- Point out: difficulty badges (easy/medium), XP rewards, time estimates
- Point out: Level/XP bar at top, date display
- Pull down to refresh

### 4. Quest Start (30s)
- Tap the first quest card
- **Quest Detail**: show description, proof hint, difficulty/XP/time metadata
- Tap **"Start Quest"**
- Note: assignment status changes to "active"

### 5. Proof & Completion (60s)
- **Active Quest Screen**: show proof capture options
- Take a photo or write a caption
- Tap **"Submit Proof"**
- Wait for upload + completion (loading spinner)
- **Reward Reveal**: animated XP breakdown appears
  - Base XP: +15 (easy quest)
  - Quality bonus: +3 (photo proof)
  - First quest of day: +5
  - Streak: +5 (day 1)
  - Total: +28 XP
- Haptic celebration fires
- Tap **"Back to Quests"**

### 6. Progress Check (30s)
- Complete one more quest (can use "Done (No Proof)" for speed)
- Switch to **Progress tab**
- Show: Level 1, XP bar filling, streak counter at 1
- If archetypes appear, show the top archetype

### 7. Daily Clear (optional, 30s)
- Complete the third and final quest
- Reward screen shows **+10 Daily Clear bonus**
- All quest cards on home are now dimmed with "Done" badge

## Key Points to Highlight

1. **No sign-up required** — anonymous auth gets user to value in < 30 seconds
2. **Real daily quests** — generated by an engine that respects vibes, difficulty curve, and cooldowns
3. **Server-authoritative progression** — XP, streaks, and levels cannot be faked client-side
4. **Proof system** — photos upload to secure storage, quality affects XP
5. **Instant reopen** — close and reopen the app, cached data shows immediately
6. **Analytics pipeline** — every key action fires PostHog events and Sentry tracks errors

## If Something Goes Wrong

| Problem | Fix |
|---------|-----|
| "No active session" error | Supabase may have restarted — restart the app |
| "Loading your quests..." stuck | Check Edge Functions are serving (`supabase functions serve`) |
| Photo upload fails | Camera permissions may be denied — use "Done (No Proof)" instead |
| No quests generated | Verify seeds ran: `psql $SUPABASE_DB_URL -c "SELECT count(*) FROM quests"` |
